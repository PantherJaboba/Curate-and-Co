## Put your images here!

This folder is for all your website images:
- Logo
- Portfolio photos
- Team photos
- Background images
- Product photos
- Client logos

## Recommended Image Sizes:

**Logo:**
- 200-300px wide
- PNG with transparent background
- Name it: `logo.png`

**Portfolio Images:**
- 1200px x 900px (4:3 ratio)
- JPG or PNG
- Optimized for web (under 500KB each)

**Hero/Background Images:**
- 1920px x 1080px
- JPG optimized
- Under 800KB

**General Tips:**
- Use descriptive names: `meadow-cafe-social.jpg` not `IMG_1234.jpg`
- Optimize images before uploading (use tinypng.com)
- JPG for photos, PNG for graphics/logos
- Keep file sizes small for fast loading
